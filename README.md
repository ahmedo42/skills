# Skills

Reusable AI agent skills for Claude Code, Cursor, Codex, and more.

## Install

```bash
# Install a specific skill
npx skills@latest add <your-github>/skills/topic-interview
npx skills@latest add <your-github>/skills/coding-interview
npx skills@latest add <your-github>/skills/system-design-interview

# Install globally
npx skills@latest add -g <your-github>/skills/topic-interview
```

## Available Skills

### Interview

- **[topic-interview](topic-interview/SKILL.md)** — 15-20 min focused technical topic interview with scorecard and debrief
- **[coding-interview](coding-interview/SKILL.md)** — 45 min FAANG-style coding interview (Python, LeetCode-style)
- **[system-design-interview](system-design-interview/SKILL.md)** — 45 min system design interview with phased structure
